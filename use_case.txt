Core Use Cases:

Our major objective would be pushing the code , obtaining logs and handling the rollbacks and inconsistencies at the utmost level.
Following are the core uses of our application :

- Push Source Code 
- Create Pull Request 
- Merge Code
- Trigger CI/CD Pipeline 
- Schedule Execution 
- Fetch Source Code
- Build Application
- Detect Build & Test Failure
- Detect Deployment Failure
- Analyze & notify Error logs
- Identify failure type
- Classify Failure (rule based)
- Classify Failure (ML based)
- Retry Failed Stage
- Restart Pipeline
